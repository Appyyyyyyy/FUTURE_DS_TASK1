# Business Sales Data Analysis

## Objective
Analyze business sales data to identify revenue trends, top-selling products, high-value categories, and regional performance.

## Project Overview
This project demonstrates business sales analysis using **Excel** and **Python**.

### Excel Analysis
- Pivot tables for top-selling products, high-value categories, and regional performance
- Revenue trend line chart
- Provides interactive data exploration and visualization within Excel

### Python Analysis
- Automates the same analysis using **pandas** for data aggregation
- Uses **matplotlib** to plot revenue trend line charts
- Calculates top products, high-value categories, and regional performance programmatically
- Enables quick updates and reproducibility without manual Excel operations

## Files
- `Business_Sales_Analysis.xlsx` – Original Excel analysis  
- `Business_Sales_Analysis.ipynb` – Python notebook replicating the Excel analysis  
- `README.md` – Project description  

## Key Insights
- **Top Products:** Contribute the highest share of total revenue  
- **High-Value Categories:** Technology generates the highest revenue  
- **Regional Performance:** West region shows the strongest sales  
- **Revenue Trends:** Sales fluctuate over time, with identifiable peaks and dips
