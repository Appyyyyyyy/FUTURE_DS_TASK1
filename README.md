# Business Sales Data Analysis

## Objective
This project analyzes business sales data to identify revenue trends, top-selling products, high-value categories, and regional performance. The goal is to derive actionable insights to support data-driven business decisions.

## Project Overview
The analysis is performed using **Microsoft Excel**, leveraging pivot tables and charts to summarize and visualize the data.

### Sheets Included:
1. **Data** – Contains all sales records along with relevant fields such as Order ID, Product Name, Category, Region, and Sales.
2. **Top-Selling Products** – Pivot table showing products contributing the highest share of total revenue.
3. **High-Value Categories** – Pivot table summarizing revenue by category; Technology is the highest revenue-generating category.
4. **Regional Performance** – Pivot table summarizing sales by region; West region shows the strongest performance.
5. **Revenue Trend** – Line chart showing sales trends over time.

## Key Insights
- Revenue trends fluctuate over time, showing patterns in sales performance.  
- A few top-selling products contribute significantly to total revenue.  
- Technology is the top revenue-generating category.  
- West region demonstrates the strongest regional sales performance.

## Tools Used
- Microsoft Excel (Pivot Tables, Charts)

## File Included
- `Business_Sales_Analysis.xlsx` – Complete Excel file with all sheets, pivot tables, and charts.
